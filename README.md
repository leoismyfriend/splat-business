# 🏠 3D Gaussian Splatting — Real Estate Business Toolkit

Turn a phone scan into a browser-based 3D property tour. No app. No VR. No agent needed.

> Built around [Luma AI](https://lumalabs.ai), [PlayCanvas SuperSplat](https://github.com/playcanvas/supersplat), and Claude.

## Quick Start (Your First Scan in 4 Steps)

1. **Record** — Open Luma AI → Capture → 3D Scene. Walk slowly, 4–8 min per room.
2. **Upload** — Luma processes automatically (~30–60 min). You get an email.
3. **Get the link** — Grab the iframe embed code from Luma's share panel.
4. **Deploy** — Drop the iframe into `client-page-template/index.html` and ship it.

## Common Mistakes

- ❌ Scanning too fast — walk at **half speed**, overlap every angle twice
- ❌ Bad lighting — open curtains, turn on every lamp, never scan at night
- ❌ Pitching the tech — lead with **outcomes** ("guests book faster, cancel less")
- ❌ Building a website before your first client — sell in person first
